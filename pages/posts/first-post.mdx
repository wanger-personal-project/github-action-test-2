# 我的第一篇博客

欢迎来到我的技术博客！这是使用 Nextra 创建的第一篇文章。

## 为什么选择 Nextra？

Nextra 是由 Vercel 团队开发的静态站点生成器，基于 Next.js。它有以下优点：

1. **简单易用** - Markdown 驱动，专注于内容创作
2. **性能优秀** - 基于 Next.js，自动代码分割和优化
3. **主题丰富** - 提供文档和博客两种主题
4. **SEO 友好** - 自动生成 meta 标签
5. **部署简单** - 一键部署到 Vercel

## 技术栈

这个博客使用了以下技术：

- **框架**: Next.js + Nextra
- **部署**: Vercel
- **CI/CD**: GitHub Actions
- **数据库**: Supabase（用于浏览量统计）
- **域名**: 自定义域名
